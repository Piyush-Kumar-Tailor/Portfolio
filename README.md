# ⚙️ Getting Started

## Clone Repository

```bash
git clone https://github.com/Piyush-Kumar-Tailor/portfolio.git
```

## Navigate to Project

```bash
cd portfolio
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
ng serve
```

Visit:

```
http://localhost:4200
```

---

# 🏗 Production Build

```bash
ng build --configuration production
```

The optimized production build will be generated inside the **dist/** directory.
